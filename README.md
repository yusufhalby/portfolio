## 🧩 Architecture Highlights

The React version of the portfolio was organized to improve maintainability by:

- Separating UI into reusable components
- Extracting project data into dedicated data files
- Moving animation logic into custom hooks
- Centralizing styling within a global stylesheet
- Keeping content separate from presentation

Custom Hooks:

- `useTypewriter` – Handles hero text typing animation
- `useFadeIn` – Handles section reveal animations using Intersection Observer

Data Sources:

- `projects.js`
- `experience.js`
- `stack.js`

```bash

```

```bash

```
